# Bookmark Manager

# User Stories

As an internet user,
So that I can quickly revisit my favourite websites,
I would like to view a list of bookmarks

Add new bookmarks

Delete bookmarks

Update bookmarks

Comment on bookmarks

Tag bookmarks into categories

Filter bookmarks by tag

Users manage their bookmarks
